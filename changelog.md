# Alone Changelog

All notable changes to the **Alone** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] **unreleased**

### Added

- Player with the ability to walk, run, jump and crouch.
- Three slots for game saves.
- Health, energy, stamina, stomach and hydration.
- Info panel that can be toggled with `i`.
- Help panel that can be toggled with `h`.
- Greeting that tells players they can press `h` for help.
- Different origins to pick from.
- Different traits to pick from.
- Large snail.
- Mussels.
- Oystercatcher nests.
- Day/night cycle with sun and moon.
- Loading screen.
- Sleep action that restores energy, press `n` to sleep.
- Primary and alternative interaction with `e` and `q`.
- Reeds that can be harvested and uprooted.
- Can eat mussels, snails, and eggs.
- Headbobbing.
- Breathing sounds when out of stamina.
- Dehydrate faster when in sun.
- In-game stats panel.
- Stats overview to character creation screen.
- Stats animation.
- Fall damage.
- Basic Inventory.
- Hearts 'lives' system.
- Ability for certain items to burn.
